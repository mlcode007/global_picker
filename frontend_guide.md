## 一、启动说明

### 前置条件

| 服务 | 状态 | 地址 |
|------|------|------|
| 后端 API | 需先启动 | `http://localhost:8000` |
| 前端 Dev Server | 本文档说明 | `http://localhost:5173` |
| MySQL | 阿里云 RDS | 自动连接 |
| Redis | 阿里云 Redis | 自动连接 |

### 启动后端

```bash
cd /Users/smzdm/global_picker/backend
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 启动前端

```bash
cd /Users/smzdm/global_picker/frontend
npm run dev
```

启动后访问：[http://localhost:5173](http://localhost:5173)

### 一键启动两个服务（后台运行）

```bash
# 启动后端
cd /Users/smzdm/global_picker/backend && uvicorn app.main:app --host 0.0.0.0 --port 8000 &

# 启动前端
cd /Users/smzdm/global_picker/frontend && npm run dev &
```

## 二、页面功能说明

### 1. 商品列表页 `/products`

**功能概览：**

| 功能 | 操作方式 |
|------|----------|
| 搜索商品 | 顶部搜索框，输入标题或店铺名，回车/点击搜索 |
| 按状态筛选 | 下拉选择：待定 / 已选 / 放弃 |
| 按区域筛选 | 下拉选择：PH（菲律宾）/ MY（马来西亚）等 |
| 排序 | 点击表头「销量」或「添加时间」列可升/降序 |
| 分页 | 底部分页器，支持每页 10/20/50/100 条 |
| 修改选品状态 | 点击状态标签（如「待定 ▾`」），下拉选择新状态 |
| 查看详情 | 点击商品标题或操作列「详情」按钮 |
| 删除商品 | 操作列「删除」，需二次确认 |
| 导出 Excel | 右上角「导出 Excel」，导出当前过滤结果全部数据 |
| 批量导入 | 右上角「批量导入」，跳转导入页 |

---

### 2. 商品详情页 `/products/:id`

页面分为左右两栏：

#### 左栏 — TikTok 商品信息

- 展示商品主图（可点击放大预览）
- 商品标题、价格（原始货币 + 折算人民币）、销量、评分、区域、店铺
- 备注填写框，填写后点击「保存备注」

#### 右栏上 — 拼多多比价

| 操作 | 说明 |
|------|------|
| 添加匹配 | 点击「添加匹配」，填写拼多多商品信息（标题、价格必填） |
| 设为主参照 | 点击匹配卡片中「设为主参照」，用于利润计算的基准采购价 |
| 计算利润 | 点击「计算利润」，自动将该商品价格填入利润计算器 |
| 删除匹配 | 点击「删除」并二次确认 |
| 主参照标识 | 蓝色边框卡片为当前主参照商品 |

#### 右栏下 — 利润计算器

| 字段 | 说明 |
|------|------|
| TikTok 折人民币 | 菲律宾售价换算后的人民币金额 |
| 拼多多采购价 | 从拼多多买入的成本价（人民币） |
| 物流成本 | 默认 ¥15，可手动修改 |
| 平台佣金率 | 默认 5%，可手动修改（如 0.05 = 5%） |

点击「计算利润」后显示：
- **TikTok 售价 / 总成本 / 利润**（三列统计卡）
- **利润率进度条**（绿色≥20%，黄色≥10%，红色<10%）
- 历史计算记录表格

#### 顶部操作栏

| 按钮 | 功能 |
|------|------|
| 状态下拉 | 快速切换选品状态 |
| 导出 | 导出该单条商品的 Excel 报表 |
| TikTok 原链接 | 新标签打开 TikTok 原始页面 |

---

### 3. 批量导入页 `/import`

#### 批量导入

1. 在文本框中**每行粘贴一个 TikTok 商品链接**
2. 页面实时显示「共 N 条链接」
3. 点击「开始导入」
4. 右侧显示导入结果（成功 N 条 / 跳过重复 N 条）
5. 点击「前往商品列表」查看

**示例链接格式：**
```
https://shop.tiktok.com/view/product/1730600050142774738?region=PH
https://shop.tiktok.com/view/product/1234567890123456789?region=MY
```

#### 单条添加

填写以下字段后点击「添加商品」：

| 字段 | 必填 | 说明 |
|------|------|------|
| TikTok 链接 | ✅ | 完整商品链接 |
| 商品标题 | 否 | 可选，后续抓取后自动覆盖 |
| 价格 | 否 | 手动填写当前售价 |
| 货币 | 否 | PHP / MYR / THB / SGD |
| 区域 | 否 | 销售区域代码 |
| 备注 | 否 | 选品备注 |

#### 报表导出

| 按钮 | 说明 |
|------|------|
| 导出全部商品 | 下载包含所有商品的 Excel 文件 |
| 前往列表筛选后导出 | 跳转列表页，筛选后点列表页导出按钮 |

## 三、Excel 导出字段说明

导出文件名：`global_picker_report.xlsx`

| 列名 | 说明 |
|------|------|
| 商品ID | 系统内部 ID |
| 商品标题 | TikTok 商品标题 |
| TikTok链接 | 原始商品链接 |
| 区域 | 销售区域（PH/MY等） |
| TikTok售价 | 原始货币价格 |
| 货币 | PHP / MYR 等 |
| 折合人民币 | 按汇率换算后的价格 |
| TikTok销量 | 累计销量 |
| 评分 | 商品评分 |
| 店铺 | TikTok 店铺名 |
| 选品状态 | pending / selected / abandoned |
| 拼多多价格(主) | 主参照商品的拼多多价格 |
| 拼多多店铺(主) | 主参照商品的拼多多店铺 |
| 利润(最新) | 最新一次利润计算结果 |
| 利润率(最新) | 最新利润率（百分比） |
| 备注 | 人工备注内容 |
| 添加时间 | 系统录入时间 |

## 四、选品工作流推荐

```
1. 批量导入链接（/import）
        ↓
2. 在商品列表（/products）查看商品
        ↓
3. 进入商品详情（/products/:id）
   ├── 手动添加拼多多同款商品及价格
   ├── 设置主参照商品
   └── 填写汇率后计算利润
        ↓
4. 根据利润率标记选品状态
   ├── 利润率 ≥ 20%  →  已选 ✅
   ├── 利润率 10-20% →  待定 ⏳
   └── 利润率 < 10%  →  放弃 ❌
        ↓
5. 导出 Excel 报表分享给团队
```
