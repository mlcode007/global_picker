## 2. Nginx 配置路径

| 路径 | 说明 |
|------|------|
| `/etc/nginx/nginx.conf` | Nginx 主配置 |
| `/etc/nginx/sites-available/global_picker` | **当前使用的站点配置** |
| `/etc/nginx/sites-enabled/global_picker` | 符号链接 → `/etc/nginx/sites-available/global_picker` |
| `/etc/nginx/sites-available/global_picker.pre-https-backup` | HTTPS 升级前备份 |
| `/etc/nginx/sites-available/global_picker.backup-20260511` | 日期备份 |
| `/root/nginx-backup-20260511-042442/` | 完整备份目录 |

### Nginx 配置要点

```nginx
# HTTP (80) - 重定向到 HTTPS
server {
    listen 80;
    server_name www.globalpicker.com globalpicker.com 47.238.72.198;
    location / { return 301 https://$host$request_uri; }
}

# HTTPS (443) - 主服务
server {
    listen 443 ssl http2;
    server_name www.globalpicker.com globalpicker.com 47.238.72.198;
    
    ssl_certificate /etc/letsencrypt/live/www.globalpicker.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/www.globalpicker.com/privkey.pem;
    
    # 前端静态文件
    root /root/global_picker/frontend/dist;
    
    # API 反向代理
    location /api/ { proxy_pass http://127.0.0.1:8000; }
    
    # 其他代理路径
    location /artifacts/ { proxy_pass http://127.0.0.1:8000; }
    location /docs { proxy_pass http://127.0.0.1:8000; }
    location /openapi.json { proxy_pass http://127.0.0.1:8000; }
    
    # Vue Router history 模式
    location / { try_files $uri $uri/ /index.html; }
}
```

## 4. 服务管理

### systemd 服务

| 服务名 | 说明 | 常用命令 |
|--------|------|----------|
| `global_picker_backend` | FastAPI 后端服务 | `systemctl restart/status global_picker_backend` |
| `nginx` | Web 服务器 | `systemctl restart/reload/status nginx` |

### 常用命令

```bash
# 后端服务
systemctl status global_picker_backend
systemctl restart global_picker_backend
systemctl stop global_picker_backend

# Nginx
systemctl status nginx
systemctl reload nginx      # 重载配置（不中断服务）
systemctl restart nginx     # 重启
nginx -t                    # 测试配置

# 证书管理
certbot certificates                    # 查看证书
certbot renew --dry-run                 # 测试续期
certbot delete --cert-name www.globalpicker.com  # 删除证书

# 定时任务
crontab -l      # 查看
crontab -e      # 编辑
```

## 6. 部署流程

### 自动化部署（从本地执行）

```bash
# 本地执行
cd /Volumes/KIOXIA/SMZDM/MAC01/项目/我的项目/global_picker
./scripts/deploy-prod.sh
```

**部署步骤**:
1. scp 同步 `backend/.env` 到服务器
2. SSH 到服务器 `git pull` 最新代码
3. `pip install -r requirements.txt` 安装后端依赖
4. `npm install && npm run build` 构建前端
5. `systemctl restart global_picker_backend` 重启后端
6. `systemctl reload nginx` 重载 Nginx

### 手动部署

```bash
ssh root@47.238.72.198
cd /root/global_picker

# 拉取代码
git pull

# 后端
cd backend
source venv/bin/activate
pip install -r requirements.txt
deactivate

# 前端
cd ../frontend
npm install
npm run build

# 重启服务
systemctl restart global_picker_backend
systemctl reload nginx
```

## 7. HTTPS 请求流程

```
用户浏览器
    |
    | http://www.globalpicker.com
    ↓
┌─────────────────────────────────┐
│  Nginx (80 端口)                 │
│  301 重定向 → https://          │
└─────────────────────────────────┘
    |
    | https://www.globalpicker.com
    ↓
┌─────────────────────────────────┐
│  Nginx (443 端口 - SSL 终止)     │
│  - SSL 证书验证                  │
│  - 安全头注入 (HSTS 等)          │
└─────────────────────────────────┘
    |
    ├── 静态文件请求 → /root/global_picker/frontend/dist/
    |
    ├── /api/* → proxy_pass → http://127.0.0.1:8000
    |                              ↓
    |                         FastAPI 后端
    |                              ↓
    |                         MySQL / Redis
    |
    ├── /artifacts/* → proxy_pass → http://127.0.0.1:8000
    ├── /docs → proxy_pass → http://127.0.0.1:8000
    └── /* → try_files → /index.html (Vue Router)
```
